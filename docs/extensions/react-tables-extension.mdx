# `ReactTableExtension`

## Summary

Add advanced tables to the editor. This extensions works only in React apps.

## Usage

### Installation

This extension is NOT installed for you when you install the main `remirror` package but need to be installed separately:

```bash
npm install @remirror/extension-react-tables
```

You can import the extension in the following way:

```ts
import { TableExtension } from '@remirror/extension-react-tables';
```
