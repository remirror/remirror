# Markdown Editor

The `<MarkdownEditor>` provides a state of the art editor to edit markdown.

```ts
import { MarkdownEditor } from '@remirror/react-editors/markdown';
```

<Basic />

Find many more examples on [Storybook](https://remirror.vercel.app/?path=/story/editors-markdown--basic).

:::note

Want to customize the behavior? Simply copy the `<MarkdownEditor>` code and change the extensions. That's what Remirror was built for.

:::note
